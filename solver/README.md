# Solver service

Astronomical image annotation by astrometry.net online api, see [online doc](http://astrometry.net/doc/net/api.html).

## Install package
npm
```bash
cd solver
npm install
```

yarn
```bash
cd solver
yarn
```

## Running the solver

`cd` the solver directory and run:

```bash
cd solver
npm start
```

or

```bash
cd solver
yarn start
```
